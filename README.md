# 📊 Student Performance Dashboard

Веб-дашборд для визуализации успеваемости студентов. Автоматическая загрузка данных из Google Sheets через Apps Script. Графики рейтинга, пропусков, рекомендаций. Фильтрация по преподавателю и подгруппе. Экспорт в CSV.

## Технологии
- JavaScript (ES6+)
- Chart.js
- Google Apps Script
- HTML5 / CSS3

## Запуск
Открой `dashboard.html` в любом браузере (Chrome, Firefox, Edge).

## Функции
- Автозагрузка данных из Google Sheets
- Столбчатые диаграммы рейтинга и пропусков
- Круговая диаграмма рекомендаций к экзамену
- Фильтрация: преподаватель (Юрча / Шаститко) → подгруппа (УИР-1 / УИР-2)
- Детальная таблица с выделением: отрицательный рейтинг (красный), пропуски >5 (бейдж), 1 место (🥇)
- Экспорт отфильтрованных данных в CSV

## Структура
- `dashboard.html` — главный файл (весь код)
- `docs/` — UML-диаграмма, блок-схемы, тест-кейсы
- `screenshots/` — скриншоты работы
